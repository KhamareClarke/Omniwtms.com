# Organization onboarding guide

This is the recommended first-hour setup flow for a new tenant (organization).

## Checklist

- Create tenant (org) in Admin
- Invite / create initial admin users
- Create at least one warehouse
- Add SKUs and initial inventory
- Add couriers (if using last-mile)
- Configure notifications (email/SMS)
- Configure billing (Stripe) if required
- Configure supplier directory + portal users (if using procurement)

## Suggested initial data

- **Warehouse**: one “Main” warehouse with basic attributes
- **SKUs**: top 20 movers + hazmat metadata if needed
- **Suppliers**: at least 1 supplier for procurement workflows

## Validate the tenant is working

- Open the dashboard
- Confirm the tenant name/branding is shown correctly
- Create a test order and confirm it appears in Orders
- (Optional) Open Empire OS dashboard and confirm recommendations render
